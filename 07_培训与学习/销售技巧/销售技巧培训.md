# 销售技巧培训

## 第一章 销售流程

### 1.1 标准销售流程

```
线索获取 → 需求挖掘 → 方案设计 → 报价谈判 → 合同签署 → 交付实施 → 售后服务
```

### 1.2 各阶段关键动作

| 阶段 | 关键动作 | 输出物 |
|------|---------|--------|
| 线索获取 | 行业展会、客户转介绍、线上推广 | 客户信息表 |
| 需求挖掘 | 深度访谈、痛点分析 | 需求调研报告 |
| 方案设计 | 方案撰写、产品演示 | 解决方案建议书 |
| 报价谈判 | 报价、商务谈判 | 报价单、合同 |
| 合同签署 | 合同评审、签署 | 正式合同 |
| 交付实施 | 项目实施、培训 | 验收报告 |
| 售后服务 | 客户成功、续费 | 续约合同 |

## 第二章 需求挖掘技巧

### 2.1 SPIN 提问法

- **S（Situation）**：了解客户现状
- **P（Problem）**：挖掘客户痛点
- **I（Implication）**：放大问题影响
- **N（Need-payoff）**：引导客户认识到价值

### 2.2 倾听技巧

- 积极倾听，不打断
- 适时提问，确认理解
- 总结归纳，验证信息

## 第三章 方案呈现技巧

### 3.1 FAB 法则

- **F（Feature）**：产品功能
- **A（Advantage）**：功能优势
- **B（Benefit）**：客户收益

### 3.2 演示技巧

- 提前了解客户关注点
- 准备同行业案例
- 强调 ROI
- 预留 Q&A 时间

## 第四章 谈判技巧

### 4.1 报价策略

- 了解客户预算
- 预留谈判空间
- 强调价值而非价格

### 4.2 异议处理

| 常见异议 | 应对策略 |
|---------|---------|
| 价格太高 | 强调 ROI 和价值 |
| 功能不够 | 展示定制能力 |
| 已有系统 | 分析现有系统不足 |
| 需要请示 | 推动高层会面 |

## 第五章 客户关系维护

### 5.1 客户分层

| 层级 | 标准 | 维护策略 |
|------|------|---------|
| A 类 | 高价值、高潜力 | 每月拜访，高层关系 |
| B 类 | 中价值 | 每季度拜访 |
| C 类 | 低价值 | 电话/邮件维护 |

### 5.2 客户成功

- 定期回访
- 价值呈现
- 续约提前布局
