# 路演材料生成规范

当用户要求生成路演/演示材料时，遵循本规范将 Markdown 内容转换为单文件自包含 HTML 演示稿。

---

## 输入格式

- 路演内容以 Markdown 撰写，`---` 分隔不同章节/幻灯片
- 每个 `##` 二级标题划分一页核心内容
- 数据表格、统计数字、论点总结均以 Markdown 格式呈现

## HTML 输出规范

生成单文件自包含 HTML，支持键盘翻页（空格/方向键）和触摸滑动。

### 设计风格

- **苹果简约风格**：系统字体（`-apple-system`, `SF Pro Display`），灰白基调，单一强调色

#### 设计 Token

| Token | 值 | 用途 |
|-------|-----|------|
| `--bg` | `#f5f5f7` | 内容页背景（浅灰） |
| `--card` | `#ffffff` | 卡片底色 |
| `--text` | `#1d1d1f` | 主文字色 |
| `--text-dim` | `#86868b` | 次要文字色 |
| `--accent` | `#0071e3` | 强调色（苹果蓝） |
| `--accent2` | `#ff6b35` | 警告/第二强调色 |
| `--red` | `#e74c3c` | 负面数据色 |
| `--green` | `#34c759` | 正面数据色 |
| `--border` | `#e5e5ea` | 边框/分割线 |
| `--radius` | `16px` | 卡片圆角 |
| `--shadow` | `0 4px 24px rgba(0,0,0,.06)` | 卡片阴影 |

#### 封面/总结页

- 背景色：`#1d1d1f`（深灰近黑）
- 文字色：`#f5f5f7`
- 大号标题居中（52px），附属信息 22px

### 幻灯片结构

- 每页一个 `.slide` div，全屏铺满（`position:absolute; inset:0`）
- 通过 `opacity` + `transform: translateY()` 实现淡入上移过渡动画（`0.45s ease`）
- 活跃页 `.active`，非活跃页 `pointer-events: none`
- **封面页**（第 1 页）：深色背景，大号标题居中
- **总结页**（最后页）：深色背景，要点回顾
- **内容页**：浅色背景，卡片承载内容块，`flex` 居中布局

### 页面元素组件

| 元素 | CSS 类 | 用法 |
|------|--------|------|
| 内容卡片 | `.card` | 白色圆角卡片，承载段落/列表/表格，max-width:860px |
| 全宽卡片 | `.card-full` | 同上，max-width:960px |
| 统计卡片 | `.card-sm` | 小型统计卡片，大号数字(44px) + 小字标签(15px) |
| 卡片行 | `.card-row` | `display:flex; gap:20px` 水平排列多个卡片 |
| 功能网格 | `.feature-grid` | 3列 CSS Grid，展示功能点（图标 + 标题 + 描述 + tag） |
| 引用块 | `.quote` | 左侧 4px 蓝色竖线，承载核心论点 |
| 大数字 | `.big-stat` | 72px 超大数字展示关键指标 |
| 图表容器 | `.chart-wrap` | Canvas 图表包装器 |

### 数据可视化（Canvas 柱状图）

使用原生 `<canvas>` 绘制柱状图，**不引入 Chart.js 等第三方库**。

#### 参数规范

```
Canvas 尺寸：880 × 380
内边距：    top:28  right:40  bottom:48  left:52

柱宽公式：  (绘图区宽度 / 数据点数) × 0.6
柱间距：    绘图区宽度 / 数据点数 × 0.4

颜色：
  柱渐变：  #0071e3 → #40a9ff（线性渐变，由下至上）
  网格线：  #e5e5ea（0.5px）
  坐标标签：#86868b（12px）

Y 轴：4 条水平网格线，标签右对齐，值由 max 到 0 等分
X 轴：每隔 3 个数据点标注年份，每 3 年显示标签
高亮：2008 / 2023 等关键年份柱顶显示数值标签，分别用 #ff6b35 和 #0071e3
```

#### Canvas 渲染时机

在 `setTimeout(100ms)` 后渲染，确保 DOM 已就绪。

### 导航系统

| 操作 | 行为 |
|------|------|
| 空格 / → / ↓ | 下一页 |
| ← / ↑ | 上一页 |
| 点击页面（非导航栏） | 下一页 |
| 移动端上滑 >40px | 下一页 |
| 移动端下滑 >40px | 上一页 |
| 点击底部圆点 | 跳转指定页 |

- **底部导航栏**：`position:fixed; bottom:0`，毛玻璃效果 `backdrop-filter: blur(20px)`
- **圆点指示器**：每个 slide 一个 `.nav-dot`（8px 圆点），活跃态变成长条 `width:24px; border-radius:4px`
- **首次交互**后隐藏提示文字（`opacity:0`）

### JavaScript 架构

```
1. 所有 slide HTML 存入数组
2. 一次性 innerHTML 渲染到容器
3. 首个 .slide 加 .active
4. goTo(idx) 统一控制：移除所有 .active → 当前 slide 加 .active → 更新圆点
5. Canvas 图表在 setTimeout 后渲染
6. 零外部依赖：无 jQuery、无 Chart.js、无任何框架
```
