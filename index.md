## 客户目录标准结构

每个成交/意向客户目录按以下子目录组织：

| 子目录 | 用途 |
|--------|------|
| `01_基础资料/` | 客户信息表、组织架构、联系人信息 |
| `02_需求分析/` | 业务需求调研报告 |
| `03_方案与报价/` | 解决方案建议书、报价单 |
| `04_招投标/` | 招标文件、投标文件 |
| `05_合同/` | 合同草案 |
| `06_沟通记录/` | 会议纪要、客户拜访记录 |
